# multi_cartoon
爬取酷漫网的所有漫画图片并存储
=
`整体的思路如下所示：`
-
        第一步：爬取到所有分类的链接地址。
        第二步：进入每一个分类链接地址，获取所有的分页地址。
        第三步：进入每一个分页地址，获取页面上cartoon的首页链接地址。
        第四步：进入cartoon首页地址，获取下面所有章节的地址。
        第五步：进入所有章节地址中，获取下面的分页地址，并解析出漫画图片地址。
        第六步：下载所有图片，进行分类保存。

    代码中没有使用成熟的爬虫框架，这样有助于更好的学习python和一些基本库的使用。
    这里只是一点皮毛，也是我第一个爬虫，希望继续努力，学习更多的知识！
   
   
展望
-
